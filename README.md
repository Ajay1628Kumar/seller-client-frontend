# Seller Client Frontend Codebase

This is the frontend codebase for the Seller Client.

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
- Shadcn UI
- React Hook Form
- React Query

## Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
